For Shadowfax.in
======================================================
======================================================

Write an endpoint to place an order. Model should emit a post save signal to calculate customers coordinates using Google geo code API.

Order table would have following fields:
		1. address line 1
		2. address line 2
		3. city
		4. contact_number
		5. Order time
		6. Amount
		7. Latitude
		8. Longitude
